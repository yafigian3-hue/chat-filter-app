# Chat Filter App 🛡️

**Chat Filter App** adalah aplikasi berbasis web yang dirancang untuk mendeteksi dan menyaring kata-kata kasar, toksik, atau tidak diinginkan dalam teks secara *real-time*. Aplikasi ini membantu pengguna memastikan pesan atau konten yang akan dikirim tetap bersih dan sopan.

## 🚀 Demo
Lihat aplikasi secara langsung di sini: [https://yafigian3-hue.github.io/chat-filter-app/](https://yafigian3-hue.github.io/chat-filter-app/)

## ✨ Fitur Utama
- **Real-Time Filtering:** Teks langsung disaring saat pengguna mengetik, mengganti kata-kata terlarang dengan simbol sensor (seperti `****`).
- **Clean UI/UX:** Antarmuka yang bersih dan minimalis untuk fokus maksimal pada teks.
- **Responsive Design:** Optimal untuk digunakan baik di perangkat desktop maupun mobile.
- **Copy to Clipboard:** Memudahkan pengguna untuk menyalin hasil teks yang sudah disaring dengan satu klik.
- **Custom Word List (Optional):** Kemampuan untuk menyesuaikan daftar kata-kata yang ingin disensor.

## 🛠️ Teknologi yang Digunakan
Projek ini dibangun menggunakan ekosistem pengembangan web modern:
- **Tailwind CSS** - Framework CSS untuk desain yang cepat, modern, dan responsif.
- **JavaScript (ES6+)** - Logik utama untuk algoritma penyaringan teks.
- **GitHub Pages** - Digunakan untuk hosting dan deployment aplikasi secara otomatis.
